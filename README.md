---

##### List of used technologies:
- Angular Framework
- webpack
---

##### To run the project locally, use:
```
git clone https://github.com/AlexMarvelo/CJSI-angular.git
cd CJSI-angular
npm i
npm run start
```
Then open *http://localhost:8080/* in your browser
